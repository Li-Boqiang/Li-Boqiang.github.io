Introduce to HPCC GNN bundle

Posted on 06-11-2023 Comments: Views:

HPCC GNN bundle provides a generalized ECL interface to Keras over Tensorflow. It provides Keras / Tensorflow operations parallelized over an HPCC cluster.

GNN.Tensor

The file defining Tensor in ECL GNN bundle, Tensor is a multi-dimensional array, which can be 1-dimensional, 2-dimensional... Currently, only REAL4-type tensors are supported. The Tensor.R4 word module manages Tensors of type REAL4.

MetaDataï¼šTensor

It is used to store Tensor-related data, such as defining the dimensions of Tensor and the type of Tensor.

Data Structure

TensorData: simply record Tensor data

Sparse type, it only records the part with value.

  • indexes: dimensions
  • value: value

Definition:

This is the most original data of Tensor, which records the data without any other management.

t_SparseDat

Sparse data used to construct Tensor.

t_Tensor

A RECORD type, similar to a structure, constructs a tensor through one of denseData and sparseData.
